# crm-platform

Frontend web platform for Smart CRM, built with Next.js.

## Current pages

- `/login`
- `/sign-in`
- `/dashboard`
- `/crm`

## Current structure

- `app/login/page.tsx`: login page
- `app/sign-in/page.tsx`: sign-in page
- `app/dashboard/page.tsx`: dashboard with tool cards
- `app/crm/page.tsx`: CRM workspace

## Run locally

```bash
npm install
npm run dev
```
